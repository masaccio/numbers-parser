numbers-parser

``numbers-parser`` is a Python module for reading and editing `Apple Numbers <https://www.apple.com/numbers/>`__\ ``.numbers`` files. It supports Numbers files generated by Numbers versions 3.x and later. It is tested against Numbers documents from 10.0 through to 14.4 (the last version of Numbers) and Numbers Creator Studio up to version 15.1, which the most current as of February 2026.

It supports and is tested against Python versions from 3.10 onwards. It is not compatible with earlier versions of Python.
